hashmv
======

Rename file to MD5 hash string.

USAGE
-----

hashmv file [ ...]

EXAMPLE
-------

  $ hashmv foobar.jpg
  foobar.jpg -> ./D41D8CD98F00B204E9800998ECF8427E.jpg

DEPENDENCY
----------

Ruby

SETUP
-----

1. Install hashmv file into some directory is defined in PATH var.
2. Add executable permission to hashmv.
